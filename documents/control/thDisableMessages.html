thDisableMessages

The thDisableMessages parameter is a flag that indicates that Thymol should make no attempt to substitute message properties or resolve message expressions. If disabled, in cases where there may be a conditional substitution of element text (e.g. th:text/th:utext), the substitution will not be performed.

Message processing is turned off if the flag is set to "true".

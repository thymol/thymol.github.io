thMessagePath

Thymol uses the thMessagePath to determine the location of properties files containing the default externalised messages for a template. These are files named "Messages.properties" or "Messages(suffix).properties" where (suffix) represents an optional locale specification in the form "_ll_CC-vv". Here, "ll" is a specifier for language, "CC" an optional country and "vv" an optional variant.

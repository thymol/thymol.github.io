thMessagePath

Thymol uses the thMessagePath to determine the location of properties files that contain the default externalised messages for a template.

The value of thMessagePath should be the location of the template file directory relative to the current template path.

By default, the externalised messages files are named "Messages.properties" or "Messages(suffix).properties".

In the second form (suffix) represents an optional locale specification in the form "_ll_CC-vv". Here, "ll" is a specifier for language, "CC" an optional country and "vv" an optional variant.

You can change the name used when searching for messages files by setting the value of the thMessagesBaseName control variable.

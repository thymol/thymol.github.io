thymol extensions

One of the most useful features of Thymol 2.0 is it's facility for defining custom extensions. You can use Thymol to prototype the custom dialects you may wish to add to Thymeleaf.

Using the extensions API you can extend the standard (Thymeleaf) dialect, you can add custom elements and attributes or add custom expression utilities.

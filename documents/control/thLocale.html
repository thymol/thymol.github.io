thLocale

The thLocale parameter sets the locale to be used during Thymol processing.

If thLocale is not set, the locale value that is used is derived from the browser's "navigator.language" variable.

If the locale is still undefined, then the default value for this parameter is used. The usual default is "en".

The default value may be set to a different value using the thDefaultLocale variable (see configuration section).

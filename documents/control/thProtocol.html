thProtocol

The thProtocol parameter sets the default protocol to apply to generated URIs.

It is often not necessary to set this value as the protocol to apply in a given template can usually be deduced by examining the current document. Deliberately setting this parameter will override the derived value.

The default value for this parameter is "file:///" which is a good choice for file-system based developments.

Using "http://" may be appropriate for Thymol-enabled Thymeleaf templates that are loaded from a web server.

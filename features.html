features

Thymol 2.0 is a JavaScript emulation of Thymeleaf 2.1.

Thymol 2.0 simulates the Thymeleaf 2.1 templating system allowing you to prototype Thymeleaf developments in a "stand alone" environment.

Users familiar with previous versions of Thymol (available on sourceforge) may recall that the pre-2.0 versions were limited to a subset of Thymeleaf 2.1 standard dialect. Thymol 2.0 fills in the gaps!

what's new in 2.0.0

  • Support for all Thymeleaf 2.1 "Expression basic objects".
  • Support for all Thymeleaf 2.1 "Expression utility objects".
  • Support for expression pre-processing.
  • Support for Thymeleaf-style message properties files.
  • Support for locale specific message processing.
  • Supports user extensions:
    • Custom attributes & custom elements - implement your own dialects!
    • Custom utility objects - functional prototyping.
  • New <script> definition "thymol.js" is now a bootstrap loader.
  • New configuration mechanism - edit "thymol.js" to set defaults.
  • Many bug fixes - many thanks to all Thymol users!
  • Improved execution speed.
  • Minimised version available.
  • Registered with bower. You can now type "bower install thymol" to add thymol to your project.
  • Now available via the excellent jsdelivr CDN!
        Use: src="http://cdn.jsdelivr.net/thymol/latest/thymol.min.js" for the latest minified version.
